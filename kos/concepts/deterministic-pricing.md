# Deterministic Pricing
Tags: #concept #pricing #deterministic
Source references: [Pricing Engine](../wiki/pricing-engine.md)

## Definition
Deterministic pricing is a rule-based quote algorithm that returns the same output for the same input every time.

## Key principles
- Parse quantity from input text consistently.
- Apply explicit cost and margin tiers.
- Round at the end of calculation.

## Derived insights
- Deterministic logic enables easier QA and protects margin assumptions against model variance.
