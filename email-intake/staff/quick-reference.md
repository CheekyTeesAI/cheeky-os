## 🎨 Stage Colors

| Color | Stage | Meaning |
|-------|-------|---------|
| 🔵 Blue | **Received** | Order is in the system, not started |
| 🟣 Purple | **Art** | Working on design/artwork |
| 🟠 Orange | **Printing** | Currently being printed |
| 🟢 Green | **Finished** | Done printing, ready to ship |
| ⚫ Gray | **Shipped** | Sent to customer |
| 🔴 Red row | **Overdue** | Past the due date — needs attention |

## 📦 Mark an Order as Shipped

1. Click the **📦 Mark Shipped** button on the dashboard
2. Type the **Order ID** (e.g. CT-20260315)
3. Type **your name**
4. Click **Mark Shipped**
5. Done — the order updates to gray/shipped

---

## 🔴 Dashboard Shows Red (Offline)

1. **Don't panic** — orders in the system are safe
2. Try clicking **↻ Refresh**
3. Check your internet connection
4. Wait 5 minutes
5. If still red → **Text Pat**
