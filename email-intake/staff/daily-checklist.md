# Cheeky Tees — Daily Checklist

> Print this page or open on your phone. Check off each item as you go.

---

## ☀️ Morning (Opening)

- [ ] Open the dashboard in your browser
- [ ] Verify the status dot is **green** (top-right corner)
- [ ] Check summary cards — note active orders and overdue count
- [ ] Review orders **due today** (sort by Due Date)
- [ ] Check Outlook inbox for new order emails
- [ ] Confirm overnight orders appear in the dashboard
- [ ] Flag any **overdue orders** (red rows) for priority
- [ ] Note any issues to report to Pat

---

## 🌤️ Midday (Production)

- [ ] Update order stages as work progresses:
  - [ ] Received → Art (when artwork is ready)
  - [ ] Art → Printing (when proof is approved)
  - [ ] Printing → Finished (when print is done + QC passed)
  - [ ] Finished → Shipped (when packaged and sent)
- [ ] Respond to any customer status questions
- [ ] Check for new orders that came in during the morning
- [ ] Refresh the dashboard to see latest data

---

## 🌙 End of Day (Closing)

- [ ] Mark all completed orders as **Shipped** (📦 button)
- [ ] Click **⬇ Export CSV** to download today's queue snapshot
- [ ] Check for remaining overdue orders — note for Pat
- [ ] Send Pat a quick end-of-day summary text:
  - How many shipped today
  - How many still in queue
  - Any issues or things to flag
- [ ] Close out for the day

---

## ⚠️ If Something Goes Wrong

- [ ] Dashboard shows **red dot** → Wait 5 min, refresh, then text Pat
- [ ] Order missing from dashboard → Check inbox, wait 10 min, then text Pat
- [ ] Customer complaint → Apologize, note details, text Pat immediately
- [ ] Not sure what to do → Text Pat — always better to ask
